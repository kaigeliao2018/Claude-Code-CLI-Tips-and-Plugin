# Contributing / 贡献指南

Thank you for your interest in contributing! This project thrives on community knowledge.

感谢你的贡献兴趣！本项目依靠社区知识共同成长。

---

## What to Contribute / 可以贡献什么

- **Tips & Tricks**: Workflows, prompt strategies, productivity hacks / 工作流、提示策略、效率技巧
- **Hooks**: Useful hook configurations with real-world use cases / 有实际用途的 Hook 配置
- **Plugins**: MCP servers, extensions, or tools you've found useful / 你觉得有用的 MCP 服务器、扩展或工具
- **Corrections**: Fix outdated or inaccurate information / 修正过时或不准确的内容

---

## How to Contribute / 如何贡献

1. **Fork** this repository / Fork 本仓库
2. **Create a branch** for your changes / 为你的改动创建分支

   ```bash
   git checkout -b feat/my-tip
   ```

3. **Make your changes** following the style guide below / 按照以下风格指南进行修改
4. **Submit a Pull Request** with a clear description / 提交 Pull Request 并附上清晰描述

---

## Style Guide / 风格指南

### Language / 语言

- All content should be **bilingual (English + Chinese)** / 所有内容应为**双语（英文 + 中文）**
- English first, Chinese below or inline / 英文在前，中文在后或并排

### Format / 格式

- Use Markdown / 使用 Markdown
- Include code examples where relevant / 在相关位置包含代码示例
- Keep tips concise and actionable / 保持技巧简洁且可操作

### Code Blocks / 代码块

Always specify the language for syntax highlighting / 始终指定语言以启用语法高亮：

````markdown
```bash
claude "explain this function"
```

```json
{
  "hooks": {}
}
```
````

| Content / 内容 | Location / 位置 |
|---|---|
| Basic tips / 基础技巧 | `tips/basic.md` |
