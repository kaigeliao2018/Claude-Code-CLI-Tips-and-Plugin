**第一款插件oh my cloud code介绍**
* **核心配置**：该插件自带7个专业AI代理，分别负责开发规划、代码编写、代码审核三类工作。

**第二款插件superpowers介绍**
* **核心定位**：该插件是提升代码质量的专用工具。
* **核心机制**：它强制开发者遵循TDD测试驱动开发流程，要求开发者在编写功能代码前先完成测试用例编写，从根源上减少代码bug。
* **使用便利性**：插件自带自动触发机制，无需开发者手动操作，即可自动执行对应流程。

* **核心特色**：插件具备独特的本能学习系统，可以自动记住开发者的使用习惯，使用时间越长，对开发者习惯的适配度越高。

**三款插件的选择建议**
* **场景化选择规则**：需要规范流程的团队开发，选择oh my cloud code；追求代码质量的个人开发，选择superpowers；追求极致开发效率的高级开发者，选择everything cloud code。
* **通用选择原则**：开发者需要根据自身的开发场景按需取用三款工具即可。
